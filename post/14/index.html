生命力

2018-05-18

今天 Eric 种的小草莓终于红透了
就当做是小草莓给 Eric 高考加油吧
不管怎么样都是自己辛辛苦苦种的呢
所以其实不是很舍得吃
但是在那里放的时间久了也会腐烂的
干脆明天回家再摘吧
Eric 一直很喜欢弄这些花花草草
因为她们是活的生物
拥有生命力
能对 Eric 的付出有所反馈
在这个世界里让 Eric 觉得温暖美好的
第一个就是自己喜欢的人
其次就是猫猫
然后就是这些花花草草
当然那些小仓鼠小兔子什么的其实也喜欢啦
第一天拿到她们的时候
还都是一株株的枯枝
又正值冬天
在阳台上把她们一个个栽好
每天浇水护理她们
寒假里基本上每天早上起床第二件事情就是去阳台
看看她们到底发芽了嘛
天气开始热起来之后
她们也开始一个个先后抽新枝了
与她们一起出现的
还有杂草和虫子
有那种绿绿的也有黑色的
绿绿的大概是蚜虫吧
反正过一段时间 Eric 就会去除草
拿小铲子把杂草铲掉
有一些不认识的杂花
如果好看的话比如一次看到了一株开蓝色花朵的植株
Eric 会小心的把她连根移植到阳台另一个角落的盆子里
尽量让她不要把种子扩散到 Eric 种的那些植物中
杂草还是比较好处理的
麻烦的是虫子
Eric 真的不知道那些虫子是哪里来的
只知道她们会吸食植物的汁液
网上买了杀虫剂
小心翼翼地喷
怕药性太强伤到土地
虫子数量有所减少
但过几天又出现了
反正按时喷药吧
实在不行去郊区抓点七星瓢虫回来养
还有之前养的水培栀子花
突然间叶子慢慢变黑
花苞也僵掉了
大概查了一下可能是得病了
什么病并不清楚毕竟 Eric 不是专业的
每天给她换水希望她能自己好起来
可是栀子花的情况一天比一天差
Eric 当时就直接跪在花瓶前哭
后来干脆一狠心把叶子全部剪了
孤零零的枝干在冬天中显得格外孤独
Eric 还是每天给她换水
栀子花还是爱 Eric 的
长出了新的嫩绿色的叶子
也没有再变黑了
还开了两朵花来谢谢 Eric
这就是为什么 Eric 喜欢这些花花草草包括小动物的原因
因为总是能从她们身上感受到生命力
她们是活的
Eric 为她们付出
她们也会给予 Eric 答复
Eric 心里一直把她们当成人
是 Eric 的朋友
喜欢的人也许不会一直陪着自己
但至少她们会
她们是有感情的
Eric 喜欢她们
并且会好好保护她们的
而且有些时候真的很羡慕她们
都有繁育后代的能力
Eric 也想要
想和自己喜欢的人创造爱情的结晶
可惜永远无法实现
和这些植物相比
Eric 才像是一株枯枝
毫无生命力
