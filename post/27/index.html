温暖

2018-06-11
这是我写的最后一篇文章了，给这个世界讲一个故事。
主人公叫 Eric，一个 18 岁的孩子，一个跨性别者，一个渴望爱的人。

她谈过两场恋爱，第一次被自己班上的一个女孩子追，那个女孩子用了半个学期的时间，进入了 Eric 的内心，知道了 Eric 是个跨性别者，那个女孩子支持她，愿意接受一切，也获得了 Eric 的信任，Eric 的第一场青涩的恋爱开始了。那个女孩子比 Eric 成熟太多，所有的一切 Eric 都处于被动。懵懂无知是最适合用来形容当时的 Eric 了，就连接吻也只是脸红着亲了一下对方的脸蛋，就像所有小情侣那样，两个人牵手，逛街，吃饭，看电影。对了，Eric 和那个女孩子看的第一场也是最后一场电影是《大鱼海棠》，那一天 Eric 哭的一塌糊涂，大概是电影里的一些情景触动了 Eric 内心深处的某个想法吧。

看过《大鱼海棠》的人应该知道这部电影的剧情吧，绚丽而又朴素的画面，配上震撼而又宁静的音乐，讲述着一个人类不知道的世界的故事。居住在神之围楼里的椿，在十六岁生日那天前往人间参加巡礼，却被渔网所困，幸运的是，她被一个人类男孩所救，不幸的是，那个男孩却从此沉入深渊。为了报恩，椿开始在这个神的世界里偷偷饲养那个男孩的灵魂，并取名为鲲。此时有另一个对椿有好感的人，叫湫。湫喜欢椿，在整个《大鱼海棠》中，湫都一直陪着椿，无私地为椿奉献着自己的一切。平静的生活还是终会被打破，因为椿在神的世界里饲养作为人类的鲲，违背了那个世界的规律而不断引发灾难，鲲被发现了，人们要求杀死鲲，还好，椿的爷爷奶奶支持椿去冒陷族人于水深火热之中，陷自己的生命于不顾的险。湫与椿的奶奶、湫的奶奶一起，帮助鲲通过海天之门，达成椿的愿望。湫的奶奶掌管海天之门，守护龙王面具，湫继承了通海天之门的能力同时也能掌管柿子树。湫天不怕地不怕，只怕椿受苦。少年为情初醉，对着天地放声宣泄自己的情感。他犯天神，偷神器，不惜为椿交出全部寿命，只为成全椿与鲲人间重逢。最后还愿化作人间风雨相伴，而这一切只是换来椿的一句谢谢以及对他像对哥哥的感情。他对椿感情如此深厚，只是情不知所起，感动也难从心起……

有的鱼是永远关不住的，因为他们属于天空。

这就是那部让 Eric 在自己的初恋面前哭的一塌糊涂的电影。并不被椿重视的湫，最后还是为了椿付出了自己的生命，只为让椿与鲲在人间重逢，这一切的背后便是爱，为了自己爱的人，即使牺牲一切，即使死在她面前也是值得的，最后只有一声谢谢也没有关系。电影终究是电影，真正愿意为了成全他人而放弃自己的幸福的人又有多少，即使装作一副毫不在乎的模样，最后真正能放下这一切的人又有多少。

至少 Eric 不是。那次之后，埋藏在 Eric 心中的种子终于发芽，在她心中开始充满了对死亡的向往，能为了爱而死是多么幸福的一件事情，只可惜这个爱已经是扭曲了的爱。

Eric 和那个女孩子分手了，女孩子给出的原因是她有自己的男神，她还是喜欢她的男神，在和 Eric 一起“谈恋爱”的时候，她也和男神一起逛街一起玩，甚至比和 Eric 在一起还要开心，至于为什么会和 Eric 一起玩呢，她说是看 Eric 太孤独，只是想陪陪Eric，并没有想要谈恋爱，也没有想到 Eric 会这么认真。于是，Eric 开始学会了如何用眉刀在自己手臂上画画，学会了如何从各种黑市上搞到精神药物，学会了如何用虚拟货币在暗网上进行违法交易，这一切只用了半年的时间。

Eric 累了，她想休息，终于她决定在她 18 岁生日那天结束一切。

有一个人找到了 Eric，他想救 Eric，想让 Eric 活下去，这个人成为了 Eric 的主人，开启了 Eric 的第二次恋爱。然后发生的一切，和第一次恋爱一样，一模一样，甚至让 Eric 怀疑自己在做梦，可惜这个世界是真实的。

Eric 想死在她喜欢的人身边，这已经是她最后的愿望了，但是每个人都知道，这个愿望其实只是 Eric 结束自己的借口，一个童话故事般的借口，你能说 Eric 在策划着自己的死亡的时候，心里没有一丝怨毒的得意和快感吗？她绝对能够预见到自己死亡的杀伤力，她倾尽了自己的生命所要达到的目标，便是希望自己被记住，甚至是报复这个世界。这目标比她的生命更有价值。如果活着做不到，那么就去死，死总是最有震慑力的。

把爱作为人生的唯一支点的人，失掉了爱，就是倾覆了整个人生，爱没有了，生的希望就没有了。她们死得那么干脆利落，毫无挂念。

很多为爱而死的人都有一个基本的共性，那就是家庭没温暖或者缺少父母的理解，她们早早的就把自己生命的价值寄托在所谓的爱情之上，爱情是他们获取温暖和存在感的方式，爱情如果改变了，那么全世界的光亮也就熄灭了。她们为爱而死，却不是死于爱，而是死于不能接受失败，死于绝望、不甘、嫉妒、怨恨……

为爱而死的人并没有抛弃这个世界，反而，在她们的心里，认为是这个世界抛弃了她们，她们想付出一切，去报复。

杜十娘怒沉百宝箱是千古绝唱，但杜十娘是死于爱吗？不，她死于绝望，一个花魁渴望遇到良人来解救自己的风尘生涯，这是她精心策划了很久很久的伟大工程，她以为李甲就是对的那个人，谁知道他不是，还卖了她，给了她一个响亮的耳光。原来她以为所有的男人都靠不住，后来她以为还有一个男人靠得住，最后她发现果然是全部的男人都靠不住，这种过山车一般的起落，彻底摧毁了一个女人想要走回良家之路的希望。她不死又如何，回去成为别人的笑柄吗？亦或是找到另外一个风尘之地，重操旧业？都不成，她见过最美的希望，几乎能够触及到自己的梦，她再也回不去了。她因李甲而死，却不是为了李甲而死。这里面有微妙的但泾渭分明的区别。

人如果不懂得如何化解自己内心的伤痛和缺陷，如果一个人对自己的生命本来就缺乏信念，就像是 Eric 这样的人，那么总有一件事，一个人，会引发生命锁链的崩塌。

爱，真正的爱，是不会死人的。因为爱会带来柔软的心境，会丰富和改变一个人的生命，这是即便分手也不能否决的美好。爱的极致甚至是放手，对方的幸福是压倒自己得失的大事。

人是社会性动物，需要温暖，需要爱，需要存在感，需要认同感，当这些元素中的某一项长时间缺失之后，人就会抑郁，当这些元素全部都消失了，人就会永远沉睡在自己的梦境之中，即使那个梦是无尽的黑暗，对她们来说也依然很美好，因为此时她们已经失去了一切。
